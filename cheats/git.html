Git Cheatsheet

Concise reference for everyday Git workflows: init, stage, commit, branch, sync, and undo safely.

Quick Commands

  • Init: git init
  • Clone: git clone <url>
  • Status: git status -s
  • Add: git add .
  • Commit: git commit -m "msg"
  • Amend (opt): git commit --amend

Branching & History

  • Create & switch: git checkout -b feat / git switch feat
  • Log compact: git log --oneline --graph
  • Diff: git diff / git diff --staged
  • Stash: git stash push -m "WIP"

Sync & Rewriting

  • Fetch: git fetch --all
  • Pull (rebase preferred): git pull --rebase
  • Push: git push origin HEAD
  • Merge: git merge feature
  • Rebase: git rebase main
  • Interactive rebase (opt): git rebase -i HEAD~3
  • Reset (danger): git reset --hard HEAD
  • Tag: git tag -a v1.0 -m "v1.0"
